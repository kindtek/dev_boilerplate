# DEV_BOILERPLATE

## to build a custom image for use in your WSL environment

### 1 - Click this link

#### <https://open.docker.com/dashboard/dev-envs?url=https://github.com/kindtek/dev_boilerplate@dev>

### 2 - Follow the prompts

#### Docker will ask to be installed on your system. Click through this and other prompts as shown

![1](blob/dev/Screenshot%202022-12-10%20015314.png)
![2](blob/dev/Screenshot%202022-12-10%20015246.png)
![3](blob/dev/Screenshot%202022-12-10%20021715.png)
![4](blob/dev/Screenshot%202022-12-10%20020509.png)

### 3 - Finally, use this terminal interface built in to Docker Desktop or use the external terminal option (Execute in CLI) to export/import the image

#### For help with that follow along with the instructions on this webpage

#### <https://learn.microsoft.com/en-us/windows/wsl/use-custom-distro>

##### BONUS: Use the "OPEN IN VS CODE" option to customize the Docker image yourself

![5](blob/dev/Screenshot%202022-12-10%20021715.png)

###### ( DOUBLE BONUS: you can also accomplish this by changing the top level domain of this (and any other) github repo url from .com to .dev - ie <https://github.dev/kindtek/dev_boilerplate/tree/dev> )
